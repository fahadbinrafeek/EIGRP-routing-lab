# EIGRP Routing Lab

## Overview
This lab demonstrates the configuration of EIGRP in Cisco Packet Tracer.

## Features
- Multiple routers configured with EIGRP
- Neighbor adjacency formation
- Automatic route exchange
- End-to-end connectivity testing (ping)

## Configuration Highlights
- Autonomous System: 100
- Network commands used for route advertisement
- Verification using:
  - show ip route
  - show ip eigrp neighbors
